# springboot-mall
基於SpringBoot、MySQL、Vue 實作商店系統


## 技術
- JAVA Spring Boot :
  - MVC架構實作後端程式
  - JDBC連接資料庫
    
- MySQL :
  - 建立資料庫
    - 商品
    - 購物車
    - 訂單
    - 用戶
    
- JWT :
  - 使用JWT實作登入驗證
 
- Vue :
  - 使用Vue作為前端框架
  - Vue Router 配置路由

- Pinia
  - 前端狀態管理及持久化
